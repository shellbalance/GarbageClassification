# GarbageClassification
一个垃圾分类识别app
#### 项目简介

------

GarbageClassification项目（以下简称App），是一个提供查询垃圾种类，识别垃圾类别的一个项目

#### 项目功能

------

- 文字手动输入功能
- 语音转文字功能
- 图片识别功能
- 浏览新闻功能
